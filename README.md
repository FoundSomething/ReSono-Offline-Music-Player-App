Resono is a kotlin based offline music player app which uses open source C codes for its audio engine.
app scans for music in your device even if its in your SDcards 
This project was my final year assesment project for my college

The app includes features like : 1. Bass Boost ( Working )
                                 2. 8-band Equalizer ( Working )
                                 3. Reverb ( its more like echo then reverb but it still works )

App has simple ui/ux and its easy to use 
 
Note: for permission open settings > apps > Resono > permissions > then allow the permission, This is needed as there is no pop-up for permission in app
